A simple console-based Car Rental System developed in Java using Object-Oriented Programming (OOP) concepts. The application allows users to view available cars, rent a car, return a rented car, and manage customer information through a menu-driven interface.

## Features

- View available cars
- Rent a car
- Return a rented car
- Manage customer details
- Track car availability
- Menu-driven console application

## Technologies Used

- Java
- Object-Oriented Programming (OOP)

## Project Structure

- Car Class - Stores car details and availability.
- Customer Class - Stores customer information.
- Rental Class - Handles car rentals and returns.
- CarRentalSystem Class - Contains the main menu and application logic.

## How to Run

1. Clone the repository.
2. Open the project.
3. Compile the Java file.
4. Run the `CarRentalSystem` class.
5. Follow the on-screen menu.
